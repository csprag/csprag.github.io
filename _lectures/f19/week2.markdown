* A Good Commit Message
    * The "Title"
        * 50 chars or less
        * Starts like a sentence but doesn't end like one
        * Imperative!
    * The "Body"
        * 72 chars or less per line
        * What and why
        * NEVER how (that's what the code is for)
        * Add any external references (eg. Fixes issue #42)
